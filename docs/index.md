# talatThaiR

**talatThaiR** เป็น R Package ที่ออกแบบมาเพื่อให้นักวิเคราะห์ข้อมูล (Data Analyst),
นักวิทยาศาสตร์ข้อมูล (Data Scientist) และนักวิจัย
สามารถดึงข้อมูลสถิติการเกษตรของประเทศไทย จาก **ศูนย์ข้อมูลเกษตรแห่งชาติ (National
Agricultural Big Data Center - NABC)** ได้อย่างง่ายดาย รวดเร็ว และเป็นระบบ

จุดเด่นของแพ็กเกจ

- **🧩 โครงสร้างมาตรฐานเดียว (Consistent API):** ทุกฟังก์ชันใช้ Parameter
  รูปแบบเดียวกัน (`category_code`, `product_code`, `date`, `start_date`,
  `end_date`) ทำให้เรียนรู้แค่ฟังก์ชันเดียวก็ใช้ได้ทั้งหมด
- **🔀 ระบบสับรางอัตโนมัติ (Smart Routing):** เพียงคุณระบุสิ่งที่คุณอยากได้ (เช่น
  หมวดหมู่ หรือ ชื่อสินค้า) ระบบจะสลับ Endpoint ของ API ไปยัง `/category`,
  `/group` หรือ `/product` ให้อัตโนมัติ
- **📖 ตารางคู่มือในตัว (Built-in Mappings):** มีฟังก์ชัน Helper
  สำหรับเรียกดูรหัสสินค้า (English Codes) หมดปัญหาการพิมพ์ชื่อสินค้าภาษาไทยผิด
  หรือเว้นวรรคไม่ตรงกับฐานข้อมูล
- **🔄 ดึงข้อมูลทะลุทุกหน้า (Auto-Pagination):** รองรับการดึงข้อมูลแบบระบุช่วงเวลา
  (`start_date` ถึง `end_date`) โดยระบบจะทำการ Loop กวาดข้อมูลทุกหน้าใน API
  มาประกอบร่างเป็น Data Frame ก้อนเดียวให้อัตโนมัติ
