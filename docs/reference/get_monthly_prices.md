# Get monthly agricultural commodity prices

Fetches monthly price data using one primary search mode, with optional
`year_th` and/or `month` filters.

**Primary search modes (choose one):**

- `category_code` — search by commodity category

- `product_code` — search by product name

- `year_th` + `month` — search by Thai year and month (when used alone)

**Optional filters (combine with category_code or product_code):**

- `year_th` — filter by Thai Buddhist year only

- `year_th` + `month` — filter by year and month

Note: `month` cannot be used without `year_th`.

## Usage

``` r
get_monthly_prices(
  category_code = NULL,
  product_code = NULL,
  year_th = NULL,
  month = NULL,
  api_key = NULL
)
```

## Arguments

- category_code:

  Category code (see
  [`show_weekly_categories()`](https://kanthjs.github.io/talatThaiR/reference/show_weekly_categories.md),
  e.g. "BUFFALO"). Monthly prices share the same category map as weekly
  prices.

- product_code:

  Product code (see
  [`show_weekly_products()`](https://kanthjs.github.io/talatThaiR/reference/show_weekly_products.md),
  e.g. "PORK_LIVE_100"). Monthly prices share the same product map as
  weekly prices.

- year_th:

  Thai Buddhist year (e.g. 2569). Can be used alone or with `month`.

- month:

  Month as integer 1-12 (e.g. 2 or "02"). Must be used together with
  `year_th`.

- api_key:

  API key (if required)

## Value

A data.frame of monthly agricultural commodity prices
