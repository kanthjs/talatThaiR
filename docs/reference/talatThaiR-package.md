# talatThaiR: Access Thai Agricultural Statistics from NABC API

Provides a consistent interface to access Thai agricultural statistics
from the National Agricultural Big Data Center (NABC) API. Supports
daily, weekly, and monthly commodity prices as well as production and
price indices at monthly, quarterly, and yearly frequencies. Features
smart endpoint routing, built-in product code mappings, and automatic
pagination.
